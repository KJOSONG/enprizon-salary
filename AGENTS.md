# Project Rules — Kilwa System

## Non-Intrusion Principle
All non-code-change work (analysis scripts, temp calculations, reports, exploration) must go into `_work/` or `/tmp/`. Never leave extra files in the project root or modify project files unless the change is intentional and user-approved.

## `_work/` Directory
- Already in `.gitignore`
- Houses temporary scripts, analysis outputs, and scratch files
- Safe to delete at any time
